Cascade — a node-based engine for generative visual coding

CFlagship
Problem

Node-graph tools for motion graphics and live visuals demand fluency in shader authoring, signal routing, and engine-specific APIs before a single frame renders — a barrier most creators never cross.

Approach

Cascade turns a natural-language brief into a typed influence graph, then a two-phase agentic pipeline — CLIP-based scene analysis, a Reasoner/Compiler/Mason chain — compiles it into a live, browser-rendered node stack in GLSL, p5.js, Three.js and WebAudio, with an autonomous repair loop that patches runtime errors.

Result

4,510 nodes generated across 384 pipeline runs with a 97% autonomous-repair approval rate. Submitted to ACM UIST '26.
